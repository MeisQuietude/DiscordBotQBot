# Discord Bot: QBot

#### version: 0.1 alpha
#### author: Quietude / MeisQuietude
#### server: https://discord.gg/hWJvzAQ

## Available Commands

> `!` - current prefics
* `help`  - dynamic help
* `user`  - info about user in server ( Name, ID, Avatar )
* `random` - get random number (0 - 100 by default)
* `role` - set role to yourself ( modify in settings )
